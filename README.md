# CODSOFT - Simple Calculator 🧮

This is my Level-1 project for the CODSOFT Web Development Internship.

## ✨ Features
- Basic arithmetic (+, −, ×, ÷)
- Decimal support
- Percent and ± toggle
- Keyboard support
- Responsive design

## 🚀 How to Run
1. Download or clone the repository.
2. Open `index.html` in any browser.
